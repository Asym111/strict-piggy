## Ключевые факты для постов
- Пропустил день — **минус 10% накоплений, безвозвратно**
- При штрафе приложение **смеётся вслух**: «ха-ха-ха» 😈
- Вывод денег — **только после достижения цели**
- Удаление аккаунта = **потеря 50% остатка**
- Сетка сумм = цифровая версия бумажного челленджа «1 000 000»
- Дуэль с другом по коду: общий «котёл позора» из штрафов
- Стрики, календарь, 10 наград, карточки для сторис
- Бесплатно, вход через Google, RU/EN
